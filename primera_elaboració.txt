⏺ El setup actual (tot fitxers estàtics, fetch() amb rutes relatives) funciona
  directament a GitHub Pages sense canvis. Passos:

  1. Inicialitza el repo i fes commit:
  cd ~/dev/manualitats
  git init
  git add *.md index.html serve.sh
  git commit -m "Recursos de manualitats amb visualitzador web"

  2. Crea el repo a GitHub:
  gh repo create manualitats --public --source=. --push
  (o --private si el vols privat)

  3. Activa GitHub Pages:
  gh api repos/{owner}/manualitats/pages -X POST -f source.branch=main -f
  source.path=/
  O manualment: Settings > Pages > Source: main / root > Save

  4. Espera ~1 minut i obre:
  https://<el-teu-usuari>.github.io/manualitats/

  Això és tot — no cal build, ni Jekyll, ni res. GitHub Pages serveix els
  fitxers estàtics directament i els fetch() relatius funcionen igual que en
